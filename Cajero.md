# Cajero Electrónico
## Descripción del proyecto
Este proyecto consiste en la simulación de un cajero electrónico desarrollado con Python. Este mismo permite al usuario realizar operaciones básicas como:

1.Consultar saldo

2. Pagar


3. Retirar

4. Depositar saldos

## Objetivo
 
Hacer uso de variables, condicionales y poner en práctica la lógica de programación

## ¿Cómo se creó?
Se hizo uso de:

1. Git y Github 
2. Python 
3. VS code

 


## Funcionalidades
Con este programa podemos hacer uso de: 
## 1. Consultar saldo

La persona que haga uso de este programa puede saber el estado en el que se encuetra su saldo.

## 2. Hacer pagos

La persona puede usar el programa con la finalidad de poder cancelar los servicios que desee.

## 3. Retiros

La persona puede usar este programa con la finalidad de retirar dinero de su cuenta.
## 4. Depositar saldo

La persona que haga uso de este programa podrá depositar saldo en su cuenta, ya que, permite al usuario ingresar una cantidad y sumarla al saldo actual.

## ¿Cómo funciona el programa?

El programa funciona a partir de:

1. Un menú de opciones

2. Condicionales 

## Estructura general 

1. Se define un saldo inicial 

2. Se muestra un menú

3. El usuario selecciona una opción

4. El sistema ejecuta dicha opción

5. Se muestra el resultado
